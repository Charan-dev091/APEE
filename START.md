# APEE — Quick Start

## Setup
1. Copy .env.example to .env
2. Add your GROQ_API_KEY
3. pip install -r requirements.txt

## Run (3 terminals)

Terminal 1 — Backend:
  python main.py

Terminal 2 — RAG Server:
  python rag_server.py

Terminal 3 — Dashboard:
  cd dashboard
  npm install
  npm run dev

## Open browser
  http://localhost:3000
